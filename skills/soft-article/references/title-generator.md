# 平台标题生成器

## 核心原则

**不同平台的标题风格差异巨大，同一主题需要生成 3-6 个不同风格的标题。**

---

## 标题公式库

### 掘金（技术深度）

**公式 1：技术实现型**
```
用 [框架/技术] 实现 [功能]：[亮点]
```
示例：
- 用 LangChain 实现 RAG：完整代码 + 性能优化
- 用 FastAPI 搭建 Claude API 代理：支持流式输出

**公式 2：深度解析型**
```
[技术] 深度解析：从 [起点] 到 [终点]
```
示例：
- Claude API 深度解析：从原理到实战
- Prompt 工程深度解析：从入门到精通

**公式 3：性能优化型**
```
[场景] 性能优化：[数量] 个 [效果] 技巧
```
示例：
- Claude API 性能优化：3 个降低延迟的技巧
- RAG 检索优化：5 个提升准确率的方法

**关键词要求**：
- 必须含技术栈（React/Vue/Python/Node.js）
- 必须含具体数字（3 个/5 种/10 倍）
- 必须含效果词（完整/深度/实战/优化）

---

### CSDN（教程步骤）

**公式 1：保姆级教程型**
```
[技术] 完整教程：从 [起点] 到 [终点]（[年份] 最新版）
```
示例：
- Claude API 完整教程：从注册到部署（2026 最新版）
- Python 调用 GPT API：从安装到实战（保姆级）

**公式 2：快速上手型**
```
[时间] 搞定 [功能]：[特点] 教程
```
示例：
- 5 分钟搞定 Claude API 接入：零基础教程
- 10 分钟搭建 AI 客服：完整代码 + 部署

**公式 3：问题解决型**
```
[问题] 解决方案：[数量] 种方法对比
```
示例：
- Claude API 超时问题解决方案：3 种方法对比
- GPT API 限流报错：5 个常见原因 + 解决方法

**关键词要求**：
- 必须含年份（2026/2025）
- 必须含时间（5 分钟/10 分钟）
- 必须含难度（零基础/保姆级/入门）

---

### 头条（科普价值）

**公式 1：是什么型**
```
[技术] 是什么？[时间] 入门指南
```
示例：
- 大模型 API 是什么？5 分钟入门指南
- RAG 技术是什么？看完这篇就懂了

**公式 2：怎么做型**
```
[场景] 怎么做？[数量] 个 [效果] 方法
```
示例：
- 大模型 API 怎么省钱？3 个实用方法
- AI 客服怎么搭建？完整流程详解

**公式 3：对比分析型**
```
[A] vs [B]：[角度] 对比分析
```
示例：
- Claude vs GPT：性能和成本全面对比
- 官方 API vs 中转服务：5 个维度分析

**关键词要求**：
- 不能用标题党（震惊/必看/最强）
- 必须科普向（是什么/怎么做/为什么）
- 必须有具体数字（3 个/5 种/10 倍）

---

### 知乎（观点深度）

**公式 1：经验分享型**
```
[问题]？我 [动作] 后的真实体验
```
示例：
- Claude API 怎么选？我用了 3 个月后的真实体验
- 大模型 API 省钱吗？我测试了 10 个方案后的结论

**公式 2：避坑指南型**
```
关于 [技术]，这 [数量] 个坑你一定要知道
```
示例：
- 关于 Claude API，这 5 个坑你一定要知道
- 关于 API 中转服务，这 3 个雷区别踩

**公式 3：深度分析型**
```
[对比/问题]：[角度] 分析
```
示例：
- Claude vs GPT：我测试了 100 次后的数据对比
- API 中转服务靠谱吗？从技术角度深度分析

**关键词要求**：
- 必须第一人称（我用了/我测试了）
- 必须有观点（真实体验/深度分析）
- 必须有数据（3 个月/100 次/10 个方案）

---

### 小红书（体验感）

**公式 1：神器推荐型**
```
[场景] 神器！[效果] [emoji]
```
示例：
- AI 写作神器！效率提升 10 倍 ✨
- 省钱神器！API 成本降低 70% 💰

**公式 2：问题解决型**
```
[问题] 终于解决了！[方法] [emoji]
```
示例：
- Claude 太贵？终于找到省钱方法了 🎉
- API 超时？这个方法亲测有效 ✅

**公式 3：入坑指南型**
```
[技术] 入坑指南：[特点] [emoji]
```
示例：
- AI API 入坑指南：新手必看 📖
- 大模型省钱攻略：实测有效 💡

**关键词要求**：
- 必须 ≤20 字
- 必须有 emoji（1-2 个）
- 必须口语化（神器/终于/亲测）

---

### 公众号（故事感）

**公式 1：场景叙述型**
```
[场景]：一个 [身份] 的 [经历]
```
示例：
- 从月花 $500 到 $50：一个开发者的降本经历
- 搭建 AI 客服的 30 天：踩坑与收获

**公式 2：转折故事型**
```
我是如何用 [技术] 解决 [问题] 的
```
示例：
- 我是如何用 API 中转服务省下 70% 成本的
- 我是如何 3 天搭建 AI 客服系统的

**公式 3：对比体验型**
```
[对比]：[时间] 使用体验分享
```
示例：
- Claude vs GPT：3 个月使用体验分享
- 官方 API vs 中转服务：真实对比

**关键词要求**：
- 必须有故事感（从...到.../我是如何）
- 必须有时间线（30 天/3 个月）
- 必须有具体身份（开发者/创业者）

## 标题生成流程

### 输入
```
主题：Claude API 成本优化
目标平台：掘金、CSDN、头条
```

### 输出
```
掘金候选标题（3 个）：
1. Claude API 成本优化实战：3 个技巧降低 70% 开销
2. 用 Python 实现 Claude API 成本监控：完整代码 + 最佳实践
3. Claude API 性能优化：从 $150/月降到 $45/月

CSDN 候选标题（3 个）：
1. Claude API 省钱完整教程：从配置到优化（2026 最新版）
2. 5 分钟搞定 Claude API 成本优化：保姆级教程
3. Claude API 费用太高？3 种降本方案对比

头条候选标题（3 个）：
1. 大模型 API 怎么省钱？3 个实用方法帮你降低 70% 成本
2. Claude API 是什么？成本优化完整指南
3. Claude vs GPT：哪个更省钱？真实数据对比
```

## 标题优化检查清单

生成后自查：
- [ ] 符合平台风格（技术/教程/科普/体验）
- [ ] 含核心关键词（技术名/场景/效果）
- [ ] 有具体数字（3 个/70%/5 分钟）
- [ ] 长度合适（掘金 30 字内，小红书 20 字内）
- [ ] 不用标题党（震惊/必看/最强）
- [ ] SEO 友好（含搜索关键词）

## 注意事项

1. **不要生成后直接用**，要根据实际内容微调
2. **标题要和内容匹配**，不能标题党
3. **同一主题的标题不能重复**，角度要不同
4. **SEO 关键词要自然**，不能堆砌
5. **平台规则要遵守**，头条/小红书最严格
